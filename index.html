DG AI · API Reference

Use DG AI as an API

Send any message to DG AI from your own app, script, or tool — and get AI-powered responses back. No authentication needed when running locally.

Base URL
/.netlify/functions/dg-ai-7616-7616
Chat Endpoint
POST /.netlify/functions/dg-ai-7616-7616
Request Body (JSON)
{
  "action": "dg",
  "message": "Build me a dark SaaS landing page"
}
Response (JSON)
{
  "reply": "Here's your dark SaaS landing page...\n```html\n..."
}
Code Examples
const res = await fetch('https://devtoolbox-api.devtoolbox-api.workers.dev/ai/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ prompt: 'Hello DG AI!', max_tokens: 200 })
});
const data = await res.json();
console.log(data.reply || data.response || data.text || data.result);
import requests

res = requests.post(
    'https://devtoolbox-api.devtoolbox-api.workers.dev/ai/generate',
    json={'prompt': 'Hello DG AI!', 'max_tokens': 200}
)
data = res.json()
print(data.get('reply') or data.get('response') or data.get('text') or data.get('result'))
curl -X POST https://devtoolbox-api.devtoolbox-api.workers.dev/ai/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt":"Hello DG AI!","max_tokens":200}'
How It Works
1
POST your message
Send a JSON request to the DG AI endpoint — no backend setup or local server required.
2
Include the action field
Set action: "dg" along with your message in the request body.
3
Receive the reply
DG AI responds with a JSON object. Access the answer via the reply field.
